## 🚀 Getting Started
### Installation
1. Install [SWI-Prolog](https://www.swi-prolog.org/Download.html)
2. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Prolog.git
   cd Prolog
   ```
3. Run any Prolog file:
   ```bash
   swipl file_name.pl
   ```
